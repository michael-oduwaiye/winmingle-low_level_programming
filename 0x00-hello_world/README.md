## Summary
This repository contains the Module 1 assignments for the WinMingle Software Engineering Training Program.  
It demonstrates foundational skills in the C programming language, including writing, compiling, assembling, and executing simple C programs.  
The exercises focus on understanding the compilation pipeline, generating assembly code, handling standard input/output, and adhering to clean coding standards.  

## Assignments

**Task 0: 0-preprocessor**  
Script that runs a C file through the preprocessor and saves the output.

**Task 1: 1-compiler**  
Script that compiles a C file but does not link.

**Task 2: 2-assembler**  
Script that generates the assembly code of a C file.

**Task 3: 3-name**  
Script that compiles a C file and creates an executable named `cisfun`.

**Task 4: 4-puts.c**  
Program that prints a message using `puts`.

**Task 5: 5-printf.c**  
Program that prints a message using `printf`.

**Task 6: 6-size.c**  
Program that prints the size of various data types.

**Task 7: 100-intel**  
Script that generates Intel syntax assembly code from a C file.

**Task 8: 101-quote.c**  
Program that prints a quote to standard error.
