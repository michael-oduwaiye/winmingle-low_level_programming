Program
WinMingle Community Software Engineering Training

Description
This directory contains solutions to Module 2 tasks of the WinMingle Software Engineering Training Program.
The module focuses on fundamental programming concepts in C, including variables, conditional statements, loops, and basic operators.

Learning Objectives

Understand and use arithmetic operators in C
Understand logical (boolean) operators
Use relational operators for comparisons
Identify TRUE and FALSE values in C
Write programs using if and if...else statements
Declare and initialize variables (char, int, unsigned int)
Print output using printf and putchar
Use while loops for repetition
Work with ASCII character values
Understand basic compiler flags like -m32 and -m64

Tasks
0. 0-positive_or_negative.c - Program that determines whether a number is positive, negative, or zero
1-last_digit.c - Program that prints the last digit of a number and evaluates it
2-print_alphabet.c - Prints lowercase alphabet using putchar
3-print_alphabets.c - Prints lowercase and uppercase alphabets
4-print_alphabt.c - Prints lowercase alphabet excluding 'e' and 'q'
5-print_numbers.c - Prints digits from 0 to 9
6-print_numberz.c - Prints digits using only putchar and no char variables
7-print_tebahpla.c - Prints lowercase alphabet in reverse
8-print_base16.c - Prints hexadecimal numbers
9-print_comb.c - Prints combinations of single-digit numbers
100-print_comb3.c - Prints combinations of two digits
101-print_comb4.c - Prints combinations of three digits
102-print_comb5.c - Prints combinations of two two-digit numbers

Compilation
All programs are compiled using:
gcc -Wall -Werror -Wextra -pedantic -std=gnu89

Notes

All code is written in the main function unless stated otherwise
No use of system() is allowed
Programs follow WinMingle C coding standards
Output format must match exactly as required
